Chapter 05. DOM 조작하기

  • 해당 엘리먼트에 마지막 자식 엘리먼트로 추가하기-append(), appendTo()
  • 존재하는 엘리먼트를 자식 엘리먼트로 추가하기
  • 해당 엘리먼트에 첫 자식 엘리먼트로 추가하기-prepend(), prependTo()
  • 해당 엘리먼트의 뒤에 원하는 내용을 추가하기-after(), insertAfter()
  • 해당 엘리먼트의 앞에 원하는 내용을 추가하기-before(), insertBefore()
  • 해당 엘리먼트를 원하는 엘리먼트로 감싸기-wrapAll(), wrap()
  • 해당 엘리먼트를 원하는 엘리먼트로 바꾸기-replaceWith(), replaceAll()
  • 해당 엘리먼트의 내용을 비우기-empty()
  • 해당 엘리먼트를 제거하기-remove()
  • 해당 엘리먼트를 복제하기-clone()
  • 해당 엘리먼트의 현재 위치와 너비 높이 알아내기-offset(), width(), height()
  • 엘리먼트의 너비 값 변경하기-width(value)
  • 엘리먼트의 높이 값 변경하기-height(value)
  • margin을 포함한 너비와 높이 값 알아내기-outerHeight()
